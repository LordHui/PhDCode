These MATLAB codes implement the variational Bayesian group-sparse
estimation methods described in the paper

   S. D. Babacan, S. Nakajima, M. N. Do, 
   "Bayesian Group-Sparse Modeling and Variational Inference," 
   submitted to IEEE Transactions on Signal Processing, 2012.

VBGS.m: Main code
ExampleRun contains an example run on a toy problem.
