This folder contains the following files:

   RunBPDN_RP.m - A script that uses D-ADMM to solve Basis Pursuit Denoising
                  with row partition. It uses networks from the file 
                  Nets_50_nodes.mat. By default, it requires the Sparco 
                  toolbox to be installed. See: 
                  http://www.cs.ubc.ca/labs/scl/sparco/                   
   
   BPDN_RP_Solver.m - Solver for D-ADMM. It uses GPSR_BB.m

   GPSR_BB.m - An algorithm provided by: www.lx.it.pt/~mtf/GPSR
